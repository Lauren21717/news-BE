Overview

Welcome to the NC News API! This API provides endpoints for managing news articles, comments, users, and topics.

Base URL: https://nc-news-api-9v2f.onrender.com/api

Response Format: All responses are in JSON format.

Topics

GET /api/topics

Returns an array of all topics.

Example Response:

{
  "topics": [
    { "slug": "mitch", "description": "The man, the Mitch, the legend" }
  ]
}
POST /api/topics

Creates a new topic.

Request Body:

{
  "slug": "coding",
  "description": "All about programming"
}

Articles

GET /api/articles

Get all articles with optional pagination, sorting, and filtering.

Query Parameters:

ParameterDescriptionDefault
sort_byColumn to sort by (e.g., votes, comment_count)created_at
orderSort order (asc, desc)desc
topicFilter by topic slug(none)
limitNumber of results per page10
pPage number1
GET /api/articles/:article_id

Get a single article by its ID.

POST /api/articles

Creates a new article.

PATCH /api/articles/:article_id

Updates an article's vote count.

Request Body:

{ "inc_votes": 1 }
DELETE /api/articles/:article_id

Deletes an article. Responds with status 204 No Content.

Comments

GET /api/articles/:article_id/comments

Gets all comments for a given article ID.

POST /api/articles/:article_id/comments

Adds a new comment to an article.

PATCH /api/comments/:comment_id

Updates a comment's vote count.

DELETE /api/comments/:comment_id

Deletes a comment. Responds with status 204 No Content.

Users

GET /api/users

Returns an array of all users.

GET /api/users/:username

Returns a single user by their username.
